GÉNÉRALITÉS

Cette page liste des enseignes où acheter des vêtements au Québec, classées par coût croissant (symboles $ à $$$$). On couvre les options économiques (friperies, rabais) jusqu’aux marques premium (manteaux d'hiver, cuir).


0. GRATUIT / RÉEMPLOI - $ -

Type Dons & réemploi : groupes communautaires, ressourceries étudiantes, Marketplace.
Courte Description Idéal pour basiques : manteaux d’occasion, foulards, bonnets, gants, vêtements de tous les jours.
Coût moyen Gratuit à symbolique ; qualité variable.


1. FRIPERIES (Renaissance, Armée du Salut, Village des Valeurs) - $ -

Type Seconde main / thrift.
Courte Description Jeans, manteaux, pulls, accessoires ; choix large selon arrivages.
Coût moyen Bas ; excellente valeur si triée avec soin.


2. AUBAINERIE / TIGRE GÉANT / WALMART - $

Type Bas prix / familles / étudiants.
Courte Description Basiques abordables : t-shirts, joggers, vêtements hiver d’entrée de gamme.
Coût moyen Bas ; rapport qualité/prix correct pour l’essentiel.


3. WINNERS / MARSHALLS - $ à $$

Type Magasins à rabais de marques.
Courte Description Trouvailles intéressantes (manteaux, chaussures, accessoires) à prix réduits.
Coût moyen Bas à modéré ; dépend des arrivages.


4. H&M / OLD NAVY / ZARA / UNIQLO* - $$

Type Basiques tendance / fast fashion.
Courte Description Essentiels de saison, parkas légères, pulls, chemises ; retours faciles.
Coût moyen Modéré ; qualité variable selon gammes.


5. SIMONS - $$

Type Grands magasins québécois (mode & maison).
Courte Description Marques maison & designers, manteaux d’hiver, tricots, accessoires.
Coût moyen Modéré à élevé ; soldes fréquentes.


6. SPORTS EXPERTS / DECATHLON - $$

Type Sport, plein air, vêtements techniques.
Courte Description Sous-couches thermiques, vestes softshell, gants techniques, chaussures.
Coût moyen Modéré ; bon rapport utilité/prix pour l’hiver.


7. LA BAIE D’HUDSON (THE BAY) - $$ à $$$

Type Grand magasin multi‑marques.
Courte Description Sélection large (travail, soirée, chaussures, accessoires).
Coût moyen Modéré à élevé ; surveiller les promotions.


8. BOUTIQUES LOCALES (TRISTAN, FRANK AND OAK, …) - $$ à $$$

Type Marques québécoises & boutiques de quartier.
Courte Description Pièces mieux finies, coupes soignées, choix responsable (matières).
Coût moyen Modéré à élevé ; durabilité supérieure.


9. MANTEAUX D’HIVER & CUIR (KANUK, RUDSAK, MACKAGE, …) - $$$ à $$$$

Type Haut de gamme / performance hiver.
Courte Description Parkas très chauds, isolation de qualité, cuir ; prix élevés mais longue durée.
Coût moyen Élevé à très élevé ; achat long terme.


10. EN LIGNE (BOUTIQUES OFFICIELLES & MARKETPLACES) - $ à $$$$

Type Sites des enseignes + marketplaces.
Courte Description Pratique pour comparer tailles/couleurs, retours ; attention aux délais et frais.
Coût moyen Tous budgets ; vérifier politiques de retour.

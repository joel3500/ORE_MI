LE VISA

Visa de résident temporaire (VRT / TRV) ou AVE (eTA)
En plus du permis d’études, vous devez possiblement un document d’entrée pour voyager au Canada : un visa (TRV) ou une autorisation de voyage électronique (eTA). Le type requis dépend de votre nationalité.
  • Vérifier ce qu’il vous faut : Ai‑je besoin d’un eTA ou d’un visa pour étudier ?
  • Trouver l’exigence selon votre passeport : IRCC : visa ou eTA
Note : si vous déposez une première demande de permis d’études depuis l’étranger, l’eTA ou le TRV est généralement traité automatiquement avec votre demande, s’il est requis.
